# Getting started

This prototype was built with [Lovable](https://lovable.dev) during [Rebuild 1](https://github.com/rebuild-europe) in Copenhagen, March 2026 — 48 hours to prototype European social platforms.

## Run locally

You need [Node.js](https://nodejs.org) installed ([nvm](https://github.com/nvm-sh/nvm#installing-and-updating) recommended).

```sh
git clone https://github.com/rebuild-europe/forest-touch-connect
cd forest-touch-connect
npm install
npm run dev
```

## Continue building

Open the project in [Lovable](https://lovable.dev) to keep iterating with AI, or work locally in any editor — changes pushed to the repo are reflected in Lovable and vice versa.

## Stack

Vite · React · TypeScript · Tailwind CSS · shadcn/ui
